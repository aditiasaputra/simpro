# Laravel 12 Vue-Inertia Master Data CRUD

Aplikasi web responsif untuk manajemen Master Data Item. Dibangun menggunakan rilis terbaru **Laravel 12 Starter Kits** dengan integrasi **Vue.js 3** dan **Inertia.js**. Antarmuka pengguna (UI) dirancang secara modern dan aksesibel menggunakan **shadcn-vue** dan **Tailwind CSS**.

Sistem ini sangat ideal digunakan sebagai kerangka dasar untuk pengembangan modul inventaris pada aplikasi Point of Sale (POS) atau sistem manajemen aset.
